# manual-testing-portfolio
Этот репозиторий показывает процесс ручного тестирования от анализа требований до оформления результатов на примере демо-магазина Sauce Demo.

## Целевой продукт

[Sauce Demo](https://www.saucedemo.com/) - учебный интернет-магазин для практики функционального тестирования.

## Структура репозитория

- `product-overview.md` - описание продукта и границы тестирования
- `01-test-plan.md` - стратегия тестирования, риски, критерии входа и выхода
