# Описание продукта

## Продукт 

Sauce Demo - демо web-приложение интернет-магазина.

## Бизнес-цель

Пользователь должен без проблем входить в систему, просматривать товары, добавлять их в корзину и оформлять заказ.

## Основные пользовательские сценарии

- открыть старницу логина
- пройти авторизацию
- просмотреть каталог товаров
- отсортировать товары по цене или названию
- открыть карточку товара
- добавить товар в корзину
- удалить товар из корзины
- перейти к оформлению
- оформить покупку
- проверить и подтвердить покупку

## Объем тестирования / Область видимости

### In-scope (что мы тестируем)

- авторизация
- каиалог товаров
- сортировка товаров
- операции с корзиной 
- оформление заказа
- сообщения валидации
- базовая навигация

### Out-of-scope ( что мы не тестируем)

- нагрузочное тестирование backend
- доступность внешних сервисов
- интеграция с платежным шлюзом
- security penetration testing (пентест)
- тестирование интеграции с платежным шлюзом

## Подхол к тестирвоанию

Проект тестируется с точки зрения пользователя с использованием функциональных, негативных и исследовательских сценариев(exploratory-сценарий)
